# Sudoku Resolver Simples

Projeto simples com interface web para resolver Sudoku.

## Como executar

```bash
python -m http.server 8000
```

Depois abra: `http://127.0.0.1:8000`

## Como usar

1. Digite os números conhecidos (1 a 9).
2. Deixe em branco as posições desconhecidas.
3. Clique em **Resolver** para ver o preenchimento e o passo a passo.
4. Clique em **Limpar** para começar outro Sudoku.
